## Исправленные проблемы:

### ❌ Ошибка "404 Page Not Found"
**Причина:** Роуты для добавления и редактирования аниме не были определены в `App.tsx`

**Решение:** Добавлены роуты:
- `/admin/anime/new` - добавление нового аниме
- `/admin/anime/edit/:id` - редактирование существующего аниме
- `/admin/site-settings` - настройки сайта

## Как использовать:

1. **Изменить название и логотип сайта:**
   - Перейдите в `/admin/site-settings`
   - Введите новое название
   - Выберите тип отображения логотипа
   - Если нужен логотип, укажите URL изображения
   - Нажмите "Сохранить настройки"

2. **Добавить новое аниме:**
   - Перейдите в `/admin/anime`
   - Нажмите "Добавить аниме"
   - Заполните форму
   - Нажмите "Добавить аниме"

3. **Редактировать аниме:**
   - Перейдите в `/admin/anime`
   - Наведите на карточку аниме
   - Нажмите кнопку редактирования
   - Внесите изменения
   - Нажмите "Сохранить изменения"

4. **Найти аниме:**
   - Перейдите в `/admin/anime`
   - Используйте поле поиска вверху страницы
   - Введите название или часть описания

5. **Управление пользователями:**
   - Перейдите в `/admin/users`
   - Используйте поиск для быстрого нахождения
   - Кликните на пользователя для просмотра деталей
   - Нажмите "Редактировать" для изменения данных
   - Можно изменить имя, email, пароль и уровень

## Технические детали:

- Все данные хранятся в `adminStore` (in-memory)
- Toast-уведомления используют `useToast` hook
- Роутинг через `wouter`
- Формы валидируются перед отправкой
- Responsive дизайн для всех экранов
